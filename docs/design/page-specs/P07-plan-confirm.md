# P07 方案确认页

## 目标

重构方案确认/订单确认页，让用户清楚看到手串预览、材料明细、优惠和应付金额。

## 上下文

- 页面实现：`miniprogram/pages/checkout/checkout.*`
- 参考图：`docs/design/references/P07-plan-confirm.png`

## 必须保留的功能

- 方案预览
- 材料清单
- 数量、尺寸、价格
- 优惠券/优惠金额
- 用户备注
- 提交订单
- 微信支付前置逻辑

## 视觉要求

- 顶部方案卡展示手串缩略图和摘要。
- 材料明细使用清晰列表。
- 底部合计与提交按钮固定在安全区上方。

## 完成标准

- 订单金额计算与后端一致。
- 备注可正常提交。
