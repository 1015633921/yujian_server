# P02 定制方式选择页

## 目标

建立“选择定制方式”页面或分流视图，让用户在五行定制、星座灵感、MBTI 定制、审美定制、自由 DIY 之间选择。

## 上下文

- 页面实现：`miniprogram/pages/custom-mode/custom-mode.*`
- 入口：`miniprogram/pages/home/home.wxml` 的“开始定制”
- 参考图：`docs/design/references/P02-custom-mode.png`

## 必须保留的功能

- 不破坏现有测算入口和自由 DIY 入口。
- 不修改支付、订单、用户数据逻辑。
- 每个模式都必须有真实跳转或明确禁用/敬请期待状态。

## 视觉要求

- 白色页面、顶部返回、标题居中。
- 模式卡片为白底细线卡，右侧使用六边形/徽章图标。
- 每个模式有标题、短说明和明确点击反馈。

## 完成标准

- 所有可用模式跳转明确。
- 不存在“看起来能点但无反馈”的入口。
