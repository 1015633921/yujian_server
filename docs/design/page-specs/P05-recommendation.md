# P05 方案推荐页

## 目标

重构方案推荐页，展示推荐方案、平衡方案和增强方案，让用户能理解方案差异并进入 DIY 或购买链路。

## 上下文

- 可能涉及：`miniprogram/pages/plan-detail/plan-detail.*`
- 可能涉及：`miniprogram/pages/recommendation-detail/recommendation-detail.*`
- 参考图：`docs/design/references/P05-recommendation.png`

## 必须保留的功能

- 推荐方案列表
- 方案详情入口
- 收藏/喜欢入口（如现有）
- 查看更多方案入口
- 真实方案数据和图片

## 视觉要求

- 顶部 tab：推荐方案、平衡方案、增强方案。
- 大卡展示主推方案图片、标签、价格。
- 次级方案用两列小卡。
- 不使用廉价电商感的大红大黄促销样式。

## 完成标准

- 方案数据真实可点。
- 图片失败有占位。
