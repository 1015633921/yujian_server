# P12 个人中心页

## 目标

重构个人中心页，让用户资料、订单、收藏、地址、优惠券和设置入口清晰可用。

## 上下文

- 页面实现：`miniprogram/pages/profile/profile.*`
- 参考图：`docs/design/references/P12-profile.png`

## 必须保留的功能

- 微信登录/资料授权
- 手机号绑定
- 头像与昵称展示
- 我的方案
- 我的订单
- 收藏
- 收货地址
- 优惠券
- 帮助与客服
- 设置/退出登录

## 视觉要求

- 顶部用户信息卡清爽，头像清晰。
- 统计数据横向展示。
- 订单状态图标统一线性风格。
- 功能列表使用白底卡片和轻分割线。

## 完成标准

- 用户资料编辑不重叠。
- 手机号授权和保存流程仍可用。
- 退出登录可用。
