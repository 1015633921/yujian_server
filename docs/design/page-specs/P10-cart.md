# P10 购物车页

## 目标

重构购物车页，让用户能清楚管理待购买方案并进入结算。

## 上下文

- 页面实现：`miniprogram/pages/inspiration-cart/inspiration-cart.*`
- 参考图：`docs/design/references/P10-cart.png`

## 必须保留的功能

- 购物车列表
- 选择/全选
- 数量调整
- 删除或管理
- 猜你喜欢
- 合计金额
- 去结算

## 视觉要求

- 商品行包含缩略图、名称、规格、价格、数量控制。
- 底部结算栏固定在安全区上方。
- “去结算”按钮深色明确。

## 完成标准

- 数量、选择和合计金额联动正确。
