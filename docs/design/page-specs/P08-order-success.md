# P08 订单提交成功页

## 目标

重构订单提交成功状态，让用户确认订单已提交并能查看订单详情。

## 上下文

- 当前可能由 `checkout` 成功态或 `order-detail` 承载，需先审计确认。
- 参考图：`docs/design/references/P08-order-success.png`

## 必须保留的功能

- 展示订单号
- 展示下单时间
- 展示预计发货/交付信息
- 查看订单详情
- 返回/继续逛入口

## 视觉要求

- 大号完成图标。
- 信息卡片简洁展示订单关键字段。
- 主按钮为查看订单详情。

## 完成标准

- 订单提交后能正确进入该状态。
- 订单号与详情页一致。
