# P09 灵感库页

## 目标

重构灵感库页，使其符合 Light Studio Lab 设计语言，同时保留社区灵感、热门推荐和筛选搜索能力。

## 上下文

- 页面实现：`miniprogram/pages/community/community.*`
- 详情页：`miniprogram/pages/community-detail/community-detail.*`
- 收藏页：`miniprogram/pages/community-favorites/community-favorites.*`
- 参考图：`docs/design/references/P09-community.png`

## 必须保留的功能

- 搜索
- 分类/标签筛选
- 灵感卡片列表
- 点击进入详情
- 收藏/浏览数据（如现有）
- 真实后台内容数据

## 视觉要求

- 顶部搜索栏轻量。
- 内容卡片使用大图+标题+人数/热度信息。
- 图片保持天然材质真实质感。
- 分类标签只使用少量功能色。

## 完成标准

- 列表、筛选、详情跳转正常。
- 没有静态假内容替代后台数据。
