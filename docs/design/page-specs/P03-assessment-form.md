# P03 测算信息填写页

## 目标

重构测算信息填写页，让出生日期、出生时间、出生地点等信息填写更清晰、可信、轻量。

## 上下文

- 页面实现：`miniprogram/pages/assessment/assessment.*`
- 参考图：`docs/design/references/P03-assessment-form.png`

## 必须保留的功能

- 出生日期选择
- 出生时间选择
- 出生地点选择
- 五行/测算相关提交接口
- 当前加载、提交和错误提示

## 视觉要求

- 顶部步骤条：基础信息、能量分析、推荐方案。
- 表单字段使用白色卡片式行项目。
- 底部主按钮固定但不能遮挡内容。
- 说明文字克制，不制造玄学夸张感。

## 完成标准

- 表单可正常提交。
- 必填项、异常项、加载状态清晰。
