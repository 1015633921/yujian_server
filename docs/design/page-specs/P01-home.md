# P01 首页视觉与结构重构

## 目标

只重构首页，使其符合 Light Studio Lab 明亮高定工作室风格，同时完整保留当前首页真实数据、跳转和入口。

## 上下文

- 页面实现：`miniprogram/pages/home/home.wxml`
- 页面逻辑：`miniprogram/pages/home/home.js`
- 页面样式：`miniprogram/pages/home/home.wxss`
- 路由配置：`miniprogram/app.json`
- 参考图：`docs/design/references/P01-home.png`
- 全局规范：`docs/design/design-system.md`

## 必须保留的功能

- 原有首页主视觉展示
- “开始定制”入口及原跳转
- 定制工坊步骤展示
- 灵感库入口
- 我的方案/历史记录入口
- 当前首页真实接口、加载状态和跳转逻辑
- 底部导航：首页、灵感、DIY、我的

## 视觉要求

- 页面整体使用暖白背景。
- 顶部主视觉使用天然材质或水晶实拍氛围。
- 黑色只作为标题、按钮和品牌锚点使用，不占据页面主体。
- 定制工坊用横向步骤轴表达 01–04。
- 下方功能区采用一张深色品牌锚点卡和多张白色功能卡。
- 卡片使用细边框、轻阴影和 12px–16px 圆角。

## 完成标准

- 首页所有原有功能可用。
- “开始定制”和各功能卡跳转正确。
- 页面没有大面积压抑黑底。
- 不影响其他页面。
