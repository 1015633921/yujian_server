# 宇涧水晶视觉设计系统

## 1. 设计定位

Light Studio Lab 明亮高定工作室。

页面以暖白和白色为主要空间，以墨黑建立结构和视觉锚点，用少量功能撞色表达分类、状态与操作反馈。黑色不能作为大面积页面背景。

## 2. 基础色

```css
--color-bg-page: #F7F6F2;
--color-bg-card: #FFFFFF;
--color-text-primary: #191919;
--color-text-secondary: #8B8881;
--color-line: #E5E2DC;
--color-anchor-dark: #20201F;
```

当前小程序已有接近色值：`#F8F6F1`、`#FBFAF7`、`#303230`、`#87947E`。后续 P00 可统一整理为全局变量。

## 3. 功能撞色

```css
--color-blue: #365C9C;
--color-yellow: #F2B51D;
--color-red: #C83B3D;
--color-purple: #8B82B3;
--color-green: #647C70;
```

使用语义：

- 蓝色：主要进度、确认、信息状态
- 黄色：新增、灵感、推荐、提示
- 红色：重要提醒、删除、异常
- 紫色：星座、审美、灵感类功能
- 绿色：保存、完成、自然、五行类功能

## 4. 色彩比例

- 暖白、白色、浅灰：约 70%
- 水晶和自然材质真实图片：约 20%
- 黑色结构与功能撞色：约 10%
- 单个页面原则上不超过两种主要撞色

## 5. 圆角与线条

- 小控件圆角：8px / 16rpx
- 普通卡片圆角：12px / 24rpx
- 重点卡片圆角：16px / 32rpx
- 胶囊按钮：999px / 999rpx
- 普通边框：1px / 1rpx
- 选中边框：1.5px–2px / 3rpx–4rpx
- 禁止使用粗重描边

## 6. 阴影

只使用极轻阴影：

```css
0 6px 24px rgba(20, 20, 20, 0.05)
```

禁止：

- 大面积黑色投影
- 浓重悬浮效果
- 玻璃拟态堆叠
- 多层高光渐变

## 7. 按钮

主按钮：

- 深色或当前页面主功能色
- 高度不低于 48px / 96rpx
- 文字清晰
- 不使用夸张渐变

次按钮：

- 白底
- 1px 浅灰边框
- 深色文字

危险操作：

- 默认保持中性
- 确认危险操作时使用红色

## 8. 图标

- 统一使用线性图标
- 推荐线宽 1.5px
- 默认深灰
- 激活状态才使用功能色
- 同一页面禁止混合多套图标风格

## 9. 图片

- 水晶图片保留真实颜色
- 主图使用自然材质、矿物、岩石、纸张或布料作为背景
- 图片构图留白充足
- 不把商品图片统一处理成黑白
- 避免廉价电商白底抠图感
- 图片失败时必须有占位状态

## 10. 动效

- 过渡时长 180ms–260ms
- 使用淡入、位移、缩放等克制动效
- 禁止持续闪烁、弹跳或夸张粒子动画
- DIY 工作台的物理动画优先保证流畅度与可用性

## 11. 小程序适配

- 页面需兼容微信小程序胶囊按钮、状态栏和底部安全区。
- 关键操作按钮不能贴边、不能被 tabBar 或安全区遮挡。
- 适配窄屏机型时优先保证主流程按钮和金额/订单信息可见。
- 列表页避免固定高度导致内容截断。
